WSExpression
============

:viewcode:`jwsexpr.spad line 1 <jwsexpr.spad#L1>` :editcode:`[edit on github] <jwsexpr.spad#L1>`


Julia Wolfram Symbolic expressions using the MathLink Julia package. It supports the Eltable category (interface) so, for example using Fibonacci polynomials fibonacci(12,jWSExpr \ :code:`x`\ ) \ :code:`=>`\  3 5 7 9 11 6 \ :code:`x`\  + 35 \ :code:`x`\  + 56 \ :code:`x`\  + 36 \ :code:`x`\  + 10 \ :code:`x`\  + \ :code:`x`\  \%.5 \ :code:`=>`\  10*x^9
