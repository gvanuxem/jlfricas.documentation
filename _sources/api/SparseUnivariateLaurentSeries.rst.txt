SparseUnivariateLaurentSeries(Coef, var, cen)
=============================================

:viewcode:`sups.spad line 1446 <sups.spad#L1446>` :editcode:`[edit on github] <sups.spad#L1446>`

* Coef: \ :ref:`Ring <l52696e67>`\ 

* var: \ :ref:`Symbol <l53796d626f6c>`\ 

* cen: Coef


Sparse Laurent series in one variable \ :ref:`SparseUnivariateLaurentSeries <l537061727365556e69766172696174654c617572656e74536572696573>`\  is a domain representing Laurent series in one variable with coefficients in an arbitrary ring. The parameters of the type specify the coefficient ring, the power series variable, and the center of the power series expansion. For example, \ ``SparseUnivariateLaurentSeries(Integer, x, 3)``\  represents Laurent series in \ ``(x - 3)``\  with integer coefficients.
