SystemSolvePackage R
====================

:viewcode:`syssolp.spad line 1 <syssolp.spad#L1>` :editcode:`[edit on github] <syssolp.spad#L1>`

* R: \ :ref:`IntegralDomain <l496e74656772616c446f6d61696e>`\ 


Symbolic solver for systems of rational functions with coefficients in an integral domain \ :code:`R`\ . The systems are solved in the field of rational functions over \ :code:`R`\ . Solutions are exact of the form variable = value when the value is a member of the coefficient domain \ :code:`R`\ . Otherwise the solutions are implicitly expressed as roots of univariate polynomial equations over \ :code:`R`\ . Care is taken to guarantee that the denominators of the input equations do not vanish on the solution sets. The arguments to solve can either be given as equations or as rational functions interpreted as equal to zero. The user can specify an explicit list of symbols to be solved for, treating all other symbols appearing as parameters or omit the list of symbols in which case the system tries to solve with respect to all symbols appearing in the input.
