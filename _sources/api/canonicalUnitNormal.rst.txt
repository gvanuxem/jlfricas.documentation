canonicalUnitNormal
===================

:viewcode:`attribs.spad line 27 <attribs.spad#L27>` :editcode:`[edit on github] <attribs.spad#L27>`


The class of all integral domains such that we can choose a canonical representative for each class of associate elements, that is \ ``associates?(a, b)``\  returns \ ``true``\  if and only if \ ``unitCanonical(a) = unitCanonical(b)``\ .
