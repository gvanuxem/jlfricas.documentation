JuliaFloat64
============

:viewcode:`julia.spad line 93 <julia.spad#L93>` :editcode:`[edit on github] <julia.spad#L93>`


\ :ref:`JuliaFloat64 <l4a756c6961466c6f61743634>`\  implements 64 bits floating point arithmetic using Julia \ ``Float64``\  type. Bear in mind that, like \ ``JuliaInt64``\ , the internal representation depends on the underlying Lisp implementation, so the usual pure arithmetic operations occur there. For other functions like sqrt, log, exp, transcendental functions etc. the calculations are performed at machine level (generally in \ ``C``\  language, or even using assembly language).
