FiniteFieldPolynomialPackage GF
===============================

:viewcode:`ffdoms.spad line 271 <ffdoms.spad#L271>` :editcode:`[edit on github] <ffdoms.spad#L271>`

* GF: \ :ref:`FiniteFieldCategory <l46696e6974654669656c6443617465676f7279>`\ 


This package provides a number of functions for generating, counting and testing irreducible, normal, primitive, random polynomials over finite fields.

:spadfun:`lexSmaller?`\: (\ :ref:`SparseUnivariatePolynomial <l537061727365556e6976617269617465506f6c796e6f6d69616c>`\  GF, \ :ref:`SparseUnivariatePolynomial <l537061727365556e6976617269617465506f6c796e6f6d69616c>`\  GF) -> \ :ref:`Boolean <l426f6f6c65616e>`\  
  \ :code:`lexSmaller?(f, g)`\  compares monic \ :code:`f`\  and \ :code:`g`\  of the same degree in the following order. Error: if \ :code:`f`\  or \ :code:`g`\  is not monic or if \ :code:`f`\  and \ :code:`g`\  have different degrees or if common degree is 0. \ :code:`f < g`\  if the number of monomials of \ :code:`f`\  is less than this number for \ :code:`g`\ . If \ :code:`f`\  and \ :code:`g`\  have the same number of monomials, the lists of exponents are compared lexicographically. If these lists are also equal, the lists of coefficients are compared according to the lexicographic ordering induced by the ordering of the elements of \ *GF*\  given by \ *lookup*\ .
