NewSparseUnivariatePolynomialFunctions2(R, S)
=============================================

:viewcode:`newpoly.spad line 207 <newpoly.spad#L207>` :editcode:`[edit on github] <newpoly.spad#L207>`

* R: \ :ref:`Ring <l52696e67>`\ 

* S: \ :ref:`Ring <l52696e67>`\ 


This package lifts a mapping from coefficient rings \ :code:`R`\  to \ :code:`S`\  to a mapping from sparse univariate polynomial over \ :code:`R`\  to a sparse univariate polynomial over \ :code:`S`\ . Note that the mapping is assumed to send zero to zero, since it will only be applied to the non-zero coefficients of the polynomial.
