:spadfun:`bits`\: % -> \ :ref:`JLInt64 <l4a4c496e743634>`\  
  \ ``bits(x)``\  returns the bit length of the mantissa of \ ``x``\ . For a result computed at prec bits of precision this can be anywhere in the range 0 \ ``<=``\  \ ``b``\  \ ``<=``\  prec. For example 0 has 0 bits, 0.75 has 2 bits, and 3.7 has 126 bits after rounding to prec = 128 (with the default rounding mode) because the two least significant bits are zero and thus get discarded.
