GuessOption

.. _l47756573734f7074696f6e-636865636b:

.. index::
   pair: GuessOption; check

:spadfun:`check`\: Union(skip, MonteCarlo, deterministic) -> % 
  \ :code:`check(d)`\  specifies how we want to check the solution. If the value is "skip", we return the solutions found by the interpolation routine without checking. If the value is "MonteCarlo", we use a probabilistic check. This option is expressed in the form \ :code:`check == d`\

:spadfun:`homogeneous`\: Union(\ :ref:`PositiveInteger <l506f736974697665496e7465676572>`\ , \ :ref:`Boolean <l426f6f6c65616e>`\ ) -> % 
  \ :code:`homogeneous(d)`\  specifies whether we allow only homogeneous algebraic differential equations. This option is expressed in the form \ :code:`homogeneous == d`\ . If \ :code:`true`\ , then maxPower must be set, too, and ADEs with constant total degree are allowed. If a PositiveInteger is given, only ADE\ :code:`'s`\  with this total degree are allowed.

:spadfun:`maxDegree`\: Union(\ :ref:`NonNegativeInteger <l4e6f6e4e65676174697665496e7465676572>`\ , arbitrary) -> % 
  \ :code:`maxDegree(d)`\  specifies the maximum degree of the coefficient polynomials in an algebraic differential equation or a recursion with polynomial coefficients. .. index::
   pair: GuessOption; maxPower

:spadfun:`maxPower`\: Union(\ :ref:`PositiveInteger <l506f736974697665496e7465676572>`\ , arbitrary) -> % 
  \ :code:`maxPower(d)`\  specifies the maximum degree in an algebraic differential equation. For example, the degree of (\ :code:`f`\ \ :code:`''`\ )\ :code:`^3`\  \ :code:`f'`\  is 4. maxPower(\ :code:`-1`\ ) specifies that the maximum exponent can be arbitrary. This option is expressed in the form \ :code:`maxPower == d`\ .

:spadfun:`maxSubst`\: Union(\ :ref:`PositiveInteger <l506f736974697665496e7465676572>`\ , arbitrary) -> % 
  \ :code:`maxSubst(d)`\  specifies the maximum degree of the monomial substituted into the function we are looking for. That is, if \ :code:`maxSubst == d`\ , we look for polynomials such that \$\ :code:`p`\ (\ :code:`f`\ (\ :code:`x`\ ), \ :code:`f`\ (\ :code:`x^2`\ ), ..., \ :code:`f`\ (\ :code:`x^d`\ ))\ :code:`=0`\ \$. This option is expressed in the form \ :code:`maxSubst == d`\ .
