  \ :code:`power_matrix(s, l, rd)`\  computes [\ :code:`m`\ , \ :code:`p`\ ] such that \ :code:`p = s^l`\  modulo reduction data \ :code:`rd`\  and column \ :code:`j`\  of \ :code:`m`\  contain \ :code:`s^j`\  modulo reduction data \ :code:`rd`\ .
