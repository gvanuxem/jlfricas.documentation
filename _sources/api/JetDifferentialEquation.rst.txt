JetDifferentialEquation(JB, D)
==============================

:viewcode:`jet.spad line 3705 <jet.spad#L3705>` :editcode:`[edit on github] <jet.spad#L3705>`

* JB: \ :ref:`JetBundleCategory <l4a657442756e646c6543617465676f7279>`\ 

* D: \ :ref:`JetBundleFunctionCategory <l4a657442756e646c6546756e6374696f6e43617465676f7279>`\  JB


\ :ref:`JetDifferentialEquation <l4a6574446966666572656e7469616c4571756174696f6e>`\  provides the basic data structures and procedures for differential equations as needed in the geometric theory. Differential equation means here always a submanifold in the jet bundle. The concrete equations which define this submanifold are called system. In an object of the type \ :ref:`JetDifferentialEquation <l4a6574446966666572656e7469616c4571756174696f6e>`\  much more than only the system is stored. \ :code:`D`\  denotes the class of functions allowed as equations. It is assumed that the \ :code:`simplify`\  procedure of \ :code:`D`\  returns only independent equations and a system with symbol in row echelon form.
