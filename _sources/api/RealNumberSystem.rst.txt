RealNumberSystem
================

:viewcode:`sf.spad line 16 <sf.spad#L16>` :editcode:`[edit on github] <sf.spad#L16>`


The real number system category is intended as a model for the real numbers. The real numbers form an ordered normed field. Note that we have purposely not included \ :ref:`DifferentialRing <l446966666572656e7469616c52696e67>`\  or the elementary functions (see \ :ref:`TranscendentalFunctionCategory <l5472616e7363656e64656e74616c46756e6374696f6e43617465676f7279>`\ ) in the definition.
