:spadfun:`laplacianMatrix`\: % -> \ :ref:`Matrix <l4d6174726978>`\  \ :ref:`Integer <l496e7465676572>`\  
  \ :code:`laplacianMatrix(s)`\  returns matrix also known as "Kirchhoff matrix" or "Admittance matrix" where: entry [\ :code:`i`\ , \ :code:`j`\ ] = inDegree(\ :code:`vi`\ ) if \ :code:`i`\  = \ :code:`j`\  (number of incoming links) \ :code:`-1`\  if \ :code:`i`\  not = \ :code:`j`\  and \ :code:`vi`\  is adjacent to \ :code:`vj`\  0 otherwise Alternatively this is defined as \ :code:`D`\  - A, where \ :code:`D`\  is the diagonal degree matrix.
