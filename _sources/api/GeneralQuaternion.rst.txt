GeneralQuaternion(R, p, q)
==========================

:viewcode:`quat.spad line 174 <quat.spad#L174>` :editcode:`[edit on github] <quat.spad#L174>`

* R: \ :ref:`CommutativeRing <l436f6d6d7574617469766552696e67>`\ 

* p: R

* q: R


\ :ref:`GeneralQuaternion <l47656e6572616c5175617465726e696f6e>`\  implements general quaternions over a commutative ring. The main constructor function is \ :spadfun:`quatern`\  which takes 4 arguments: the real part, the \ ``i``\  imaginary part, the \ ``j``\  imaginary part and the \ ``k``\  imaginary part.
