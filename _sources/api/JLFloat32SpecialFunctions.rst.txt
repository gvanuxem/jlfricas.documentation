JLFloat32SpecialFunctions
=========================

:viewcode:`jf32sf.spad line 1 <jf32sf.spad#L1>` :editcode:`[edit on github] <jf32sf.spad#L1>`


Special functions computed using Julia\ :code:`'s`\  ecosystem. They are here essentially for "completeness" purpose with \ :code:`JLFloat32`\ . You should use the DoubleFloat\ :code:`'s`\  special functions if available, calling Julia functions is costly.
