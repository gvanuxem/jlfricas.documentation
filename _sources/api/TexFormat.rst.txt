TexFormat
=========

:viewcode:`tex.spad line 1 <tex.spad#L1>` :editcode:`[edit on github] <tex.spad#L1>`


\ :ref:`TexFormat <l546578466f726d6174>`\  provides a coercion from \ :ref:`OutputForm <l4f7574707574466f726d>`\  to TeX format. The particular dialect of TeX used is LaTeX. The basic object consists of three parts: a prologue, a tex part and an epilogue. The functions \ :spadfun:`prologue`\ , \ :spadfun:`tex`\  and \ :spadfun:`epilogue`\  extract these parts, respectively. The main guts of the expression go into the tex part. The other parts can be set (\ :spadfun:`setPrologue!`\ , \ :spadfun:`setEpilogue!`\ ) so that they contain the appropriate tags for printing. For example, the prologue and epilogue might simply contain \ :code:````\ \verb+\ :code:`\[`\ +\ :code:`''`\  and \ :code:````\ \verb+\ :code:`\]`\ +\ :code:`''`\ , respectively, so that the TeX section will be printed in LaTeX display math mode.
