.. _l494c6f676963-646564756374696f6e73:

.. index::
   pair: ILogic; deductions

:spadfun:`deductions`\: \ :ref:`List <l4c697374>`\  % -> \ :ref:`List <l4c697374>`\  % 
  assumes \ :code:`ln`\  contains a list of factors which must be \ :code:`true`\  for the whole to be \ :code:`true`\  (such as the list produced by factor). From this deductions attempts to produce a list of other proposition that must also be \ :code:`true`\  by using modus ponens. This is used to determine the returned type when converting ILogic to types by using the Curry-Howard isomorphism.

.. _l494c6f676963-666163746f72:

.. index::
   pair: ILogic; factor

:spadfun:`factor`\: % -> \ :ref:`List <l4c697374>`\  % 
  splits \ :code:`n`\  into a list of factors which must be \ :code:`true`\  for the whole to be \ :code:`true`\ . This assumes that the top level is already a set of factors separated by \ :code:`/\`\  otherwise the result will just be a list with one entry: \ :code:`'n'`\ . This is used when converting ILogic to types by using the Curry-Howard isomorphism.
