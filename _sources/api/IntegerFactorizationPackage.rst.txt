IntegerFactorizationPackage I
=============================

:viewcode:`intfact.spad line 310 <intfact.spad#L310>` :editcode:`[edit on github] <intfact.spad#L310>`

* I: \ :ref:`IntegerNumberSystem <l496e74656765724e756d62657253797374656d>`\ 


This Package contains basic methods for integer factorization. The factor operation employs trial division up to 10, 000. It then tests to see if \ :code:`n`\  is a perfect power before using Pollards rho method. Because Pollards method may fail, the result of factor may contain composite factors. We should also employ Lenstra\ :code:`'s`\  elliptic curve method.
