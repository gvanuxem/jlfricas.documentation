JuliaWSReal¶

jws.spad line 411 [edit on github]

Julia Wolfram Symbolic real numbers using Wolfram Symbolic Transport Protocol. Precision is determined during coercion. Machine precision will be used for jWSReal("3.14”) whereas jWSReal("3.14159265358979323846264338327950288419716939937510”) will have arbitrary precision of about 50 digits.
